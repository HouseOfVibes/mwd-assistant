# MWD Assistant Knowledge Base

This directory contains the authoritative reference information that the MWD Assistant uses to answer questions and provide accurate information about MW Design Studio.

## Contents

| File | Description | Last Updated |
|------|-------------|--------------|
| [SERVICES.md](SERVICES.md) | Complete catalog of services, pricing, and availability | November 23, 2025 |

## Purpose

The knowledge base provides:

- **Accurate Information**: Single source of truth for services, pricing, and policies
- **Consistent Responses**: Ensures the assistant provides uniform answers
- **Easy Updates**: Centralized location for maintaining current information

## Usage

The MWD Assistant references these files when:

- Answering questions about services and pricing
- Providing service recommendations
- Explaining availability and service areas
- Discussing discontinued services or upcoming features

## Maintenance

### Updating Existing Files

1. Update the relevant markdown file
2. Update the "Last Updated" date in the file header
3. Add an entry to the Version History section
4. Commit with a descriptive message

### Adding New Knowledge

When adding new knowledge files:

1. Create a new markdown file in this directory
2. Include a header with version and last updated date
3. Update this README with the new file entry
4. Follow the formatting conventions of existing files

## File Naming Convention

- Use UPPERCASE for primary knowledge files (e.g., `SERVICES.md`, `POLICIES.md`)
- Use descriptive names that indicate content
- Keep names concise but clear
